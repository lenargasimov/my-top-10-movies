# Installation on Linux and Unix like OS

```
git clone https://github.com/lenargasimov/my-top-100-movies.git
cd my-top-100-movies
pip3 install -r requirements.txt 
python3 main.py
```

# Installation on Windows

```
git clone https://github.com/lenargasimov/my-top-100-movies.git
cd my-top-100-movies
```
